How to Join LITSL.NET

Java Edition

1. Open Minecraft Java Edition.

2. Click on "Multiplayer".

3. Click on "Add Server".

4. Enter "litsl.net" as the server address.

5. Click "Done" and join the server.

Bedrock Edition

1. Open Minecraft Bedrock Edition.

2. Click on "Play".

3. Go to the "Servers" tab.

4. Scroll down and click on "Add Server".

5. Enter "litsl.net" as the server address and "19132" as the port.

6. Click "Save" and join the server.
